# Boredom & Dungeons — Clean Core V95

זו גרסת תיקון כיוון תנועה אחרי V94.

## מה תוקן ב־V95

ב־V94 העכבר שינה את כיוון התנועה לכל מקשי התנועה.

הבקשה עכשיו:

```text
אחורה לא צריך להשתנות לפי העכבר
רק קדימה
```

ב־V95:

```text
W / Up    → מושפע מהעכבר בתוך 60° קדימה
S / Down  → לא מושפע מהעכבר
A / Left  → לא מושפע מהעכבר
D / Right → לא מושפע מהעכבר
```

## ההתנהגות עכשיו

```text
W קדימה:
    לפי הכיוון שהעכבר הטה בתוך front cone של 60°

S אחורה:
    אחורה לפי המצלמה/המסך, לא לפי העכבר

A/D:
    שמאלה/ימינה לפי המצלמה/המסך, לא לפי העכבר
```

## גם על הסוס

בזמן רכיבה:

```text
W / Up    → מושפע מה־mounted mouse aim בתוך 60°
S / Down  → אחורה לפי המצלמה/מסך
A / D     → צדדים לפי המצלמה/מסך
Q         → יורה לפי הכיוון המוגבל
חרב       → עדיין חסומה על הסוס
```

## התחמקות

```text
Double W → מושפע מהכיוון הקדמי/עכבר בתוך 60°
Double S → אחורה לפי מצלמה/מסך
Double A → שמאלה לפי מצלמה/מסך
Double D → ימינה לפי מצלמה/מסך
```

## נשמר מ־V94

- העכבר משנה את כיוון המודל בתוך 60°.
- W קדימה יכול לסטות לפי העכבר.
- העכבר לא יכול לסובב אחורה.
- סוס פונה לשחקן רק כשהוא בא אליו.
- חרב לא מפספסת אויבים צמודים.
- Dodge effect הוא 2D ground trail.
- חיווי שאפשר לרפא את הסוס.
- אין חרב על הסוס, רק ירי.
- קליעים פוגעים בקירות.
- Enemy attack telegraphs.
- Ranged muzzle flash/trail/impact.
- Enemy HP bars.
- Death reset.

## התקנה נקייה

מחק לגמרי:

```text
Assets/_Project
```

ואז העתק מה־ZIP את:

```text
Assets/_Project
```

חכה ל־Compile מלא ואז בדוק:
- W עם עכבר ימינה/שמאלה.
- S עם עכבר ימינה/שמאלה — לא אמור לשנות כיוון לפי העכבר.
- A/D גם לא אמורים להיגרר לפי העכבר.
- אותו דבר על הסוס.
