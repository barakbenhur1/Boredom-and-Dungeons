# Boredom & Dungeons — Clean Core V16

זו גרסה נקייה מלאה להחלפה, לא פאטץ׳.

## למה V16

ב־V15 המבוך היה פתיר ועבד, אבל היה קל מדי פשוט לרוץ ליציאה:

- היו מעט מדי אויבים יחסית לגודל המבוך.
- לא בכל חדר היה Patrol Guard שיכול לחסום יציאה בגוף.
- אזורי ניסיון היציאה היו צרים מדי, אז האויבים התחילו להגיב מאוחר מדי.

## מה השתנה

### 1. הרבה יותר אויבים במבוך

מחולל המבוך עכשיו מייצר הרבה יותר חדרי קרב:

- חדרים קרובים להתחלה עדיין קלים יותר.
- רוב החדרים בהמשך מקבלים אויבים.
- חדרים קרובים לסוף מקבלים יותר אויבים.
- תא היציאה העליון מקבל Guard pressure, כדי שלא יהיה קל פשוט לרוץ לסוף.

### 2. כמעט בכל חדר קרב יש Patrol Guard

כל חדר קרב מקבל קודם כל:

```text
BD_Enemy_PatrolGuard
```

זה חשוב כי ה־Patrol Guard הוא האויב הייעודי שחוסם יציאות בגוף שלו.

אחרי זה נוספים אויבים נוספים לפי עומק החדר במבוך:

```text
Sword
Charger
TrapLayer
RangedShooter
Jumper
```

### 3. זיהוי ניסיון יציאה מוקדם יותר

אזורי ה־Exit Pressure הוגדלו:

```text
ExitPressureDepth = 9.5
ExitPressureWidth = DoorWidth + 5
```

כלומר האויבים מתחילים להגיב כשאתה מתקרב לכיוון היציאה, לא רק כשאתה כבר עומד בתוך הפתח.

### 4. Patrol Guards חוסמים יציאה בצורה חזקה יותר

ה־Patrol Guard בזמן ניסיון יציאה:

- רץ מהר יותר לפתח.
- עומד קרוב יותר לנקודת היציאה.
- מקבל radius זמני גדול יותר כדי שהחסימה בגוף תהיה מורגשת.
- תוקף מהר יותר את השחקן או את הסוס כשהם מנסים לעבור.

## נשמר מ־V15

- השחקן לא נמתח כשהוא עולה על הסוס.
- קירות שמסתירים את השחקן נהיים חצי שקופים.
- הקירות גבוהים.
- הסוס נפגע.
- הסוס זורק את השחקן אחרי רצף פגיעות.
- קפיצת סוס.
- ירידה מהסוס לפי כיוון הקלט.
- אין קירות/קוליידרים מלאכותיים שחוסמים פתחים.

6. עדיין אפשר לברוח אם משחקים טוב, אבל זה כבר לא אמור להיות קל.
