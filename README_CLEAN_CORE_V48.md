# Boredom & Dungeons — Clean Core V48

זו גרסה נקייה מלאה להחלפה, לא פאטץ׳.

## מה תוקן ב־V48 — Mouse Point Aim

ב־V48 אין יותר Cursor Lock ואין יותר cumulative yaw.

השליטה עכשיו היא:

```text
Mouse position = כיוון הסתכלות / כיוון מכה / כיוון ירי
Player idle = המודל מסתובב במקום לפי העכבר, אבל לא זז
Player moving = השחקן הולך לכיוון שהעכבר מצביע
Sword / Heavy / Q = יוצאים לכיוון העכבר
```

## התנהגות מדויקת

```text
לא לוחץ תנועה:
    המודל מסתובב במקום לפי העכבר
    המצלמה נשארת מאחורי כיוון המודל
    חרב/ירי יוצאים לכיוון העכבר

לוחץ W/A/S/D או חצים:
    השחקן הולך לכיוון שהעכבר מצביע
    המודל מסתכל לכיוון ההליכה
    המצלמה מסתדרת מאחוריו
```

## על הסוס

```text
לא זז:
    הסוס/רוכב מסתובבים במקום לפי העכבר
    Q יורה לכיוון העכבר

זז:
    הסוס הולך לכיוון שהעכבר מצביע
```
