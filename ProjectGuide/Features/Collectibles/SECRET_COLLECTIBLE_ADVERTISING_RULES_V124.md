# Secret Collectible Advertising Rules V124

## Core rule

Secret collectibles are discovered through play, not through instructions.

## Forbidden in gameplay UI

```text
Collect the Game Boy
Find the Game Boy
Collect batteries
Find batteries
Collect the cartridge
Find the cartridge
Missing Game Boy
Missing Battery
Missing Cartridge
0/4, 1/4, 2/4, 3/4, 4/4
Quest checklist
Objective marker
Arrow to collectible
```

## Allowed

```text
Badge after pickup
Pickup visual
Pickup sound later
Environmental hint
Cinematic consequence
```

## Current allowed badge labels

```text
GB
BAT
BAT x2
CART
```

These are allowed only after the player has actually collected the item.
