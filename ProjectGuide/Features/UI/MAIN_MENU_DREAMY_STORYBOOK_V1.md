# Dreamy Childhood Adventure Main Menu V1

The main menu should feel like the opening page of a childhood adventure:
dreamy, mysterious but safe, handcrafted, and connected to bedrooms,
imagination, games, mazes, and distant adventure.

## Visual system

- Procedural midnight-indigo sky.
- Warm moon glow.
- Softly twinkling stars.
- Drifting lavender cloud haze.
- Layered blue silhouettes.
- A small winding golden path leading into the adventure.
- A translucent midnight-blue storybook panel with warm-gold trim.
- Warm ivory title text with cool blue depth.
- Deep-blue buttons with gold framing and brighter hover states.

## Existing behaviour

The menu keeps `START GAME`, `SETTINGS`, desktop `QUIT`, pause/settings flow,
and the permanent lit Game Boy relic after Mother victory.

## Font safety

The menu and BBH intro use Unity's current GUI skin fonts. They do not create,
retain, or destroy temporary operating-system fonts.
