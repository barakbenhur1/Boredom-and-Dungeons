
## ModernHandheld/Controls

`Assets/_Project/Resources/ModernHandheld/Controls/` contains transparent top-surface textures cropped from the approved orthographic source sheet. These textures may only be applied to modeled moving control parts; they are not a replacement for 3D geometry. X/Y/A/B, each D-pad segment and the two center buttons have dedicated or shared cap assets with matching Unity import settings.
