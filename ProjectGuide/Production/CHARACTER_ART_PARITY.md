# Boy/Girl Character Artwork Parity

Every UI, menu, loading, current-run, next-up, tutorial, progression, promotional or preview image that contains the Boy must include a parallel Girl version in the same change.

The pair must match in dimensions, crop, camera, composition, background, lighting, horse pose when present, grading, safe areas, file format, naming relationship and import settings. Only the player character changes.

Recommended names:

```text
<AssetName>_Boy_V###
<AssetName>_Girl_V###
```

Runtime selection is deterministic from active character identity. Random selection or a Boy image on the Girl route is invalid. A missing pair blocks completion.
