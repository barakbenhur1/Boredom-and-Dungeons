# Roadmap Documentation

The single authoritative complete development plan and progress tracker is:

`/PROJECT_STATUS.md`

Rules:

- Update `/PROJECT_STATUS.md` in every material commit that changes code, requirements, QA truth, priority, blockers, or implementation order.
- Previous / Current / Next are maintained at the top of that file.
- Dedicated design documents may contain deeper specifications, but they do not replace the full authoritative checklist.
- `PROJECT_STATUS_CURRENT*.md`, `WORKING_NOW.md`, and the old V128 stage order are not live sources of truth.
- `MASTER_REMAINING_WORK_ROADMAP_V128.md` remains historical only.
