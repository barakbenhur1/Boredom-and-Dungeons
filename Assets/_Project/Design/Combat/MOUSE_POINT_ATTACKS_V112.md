# Mouse Point Attacks V112

## Goal

Attacks should go where the mouse points.

Movement rules are not changed.

## Changed

- Light melee attack uses mouse point direction.
- Heavy melee attack uses mouse point direction.
- Ranged attack uses mouse point direction.
- Mounted ranged attack uses mouse point direction.
- When the player attacks, the player turns to face the attack direction.
- When mounted and shooting, the horse/body turns to face the shot direction at the attack moment.

## Not changed

- WASD movement.
- Dodge rules.
- Camera.
- Mouse movement/front-cone model rules.
- Collectible placement.
- HUD.
- Enemies.
- Map.

## Rule

Movement can stay under the current movement design, but combat aim is direct mouse-world aim.
