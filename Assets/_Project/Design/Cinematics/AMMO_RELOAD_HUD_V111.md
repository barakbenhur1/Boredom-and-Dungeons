# Ammo Reload HUD V111

## Goal

The player needs a professional readable indicator for:
- bullets remaining
- reload time remaining
- animated recharge state

## Added

A separate bottom-right ranged ammo widget:
- ammo number
- ammo pips
- animated reload bar
- animated radial reload ring
- Ready / Reload seconds state

## Not changed

- ammo balance
- ranged damage
- reload duration
- controls
- combat logic
