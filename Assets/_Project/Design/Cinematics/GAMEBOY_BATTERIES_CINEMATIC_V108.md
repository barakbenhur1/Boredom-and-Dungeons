# Game Boy + Batteries Collectible Cinematic V108

## Added flow

1. Player collects Game Boy.
2. Player collects batteries.
3. Player enters the dark room.
4. If the player has the Game Boy and 2 batteries, the cinematic starts.
5. Player walks to the chair.
6. Player sits.
7. Player pulls out the Game Boy.
8. Batteries move into the Game Boy.
9. Screen turns green.
10. Player plays for a short moment.

## Current implementation

This is a procedural prototype cinematic:
- no external animation assets yet
- no final model assets yet
- no HUD text
