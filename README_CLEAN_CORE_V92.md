# Boredom & Dungeons — Clean Core V92

זו גרסת תיקון סוס/קרב/התחמקות אחרי V91.

## מה תוקן ב־V92

### 1. סוס פונה לשחקן רק כשהוא בא אליו

ההתנהגות עכשיו:

```text
אם הסוס רחוק ומחליט לבוא לשחקן
→ הסוס זז לכיוון השחקן ופונה אליו

אם השחקן מתקרב לסוס
→ הסוס לא משנה תנוחה רק בגלל שהשחקן הגיע אליו

אם הסוס בטווח נוחות
→ הוא עומד ולא עושה tracking rotation
```

### 2. התקפת חרב לא מפספסת אויבים קרובים מדי

לפני V92 בדיקת המכה הייתה בעיקר sphere קדמי בקצה הטווח.  
אם האויב היה ממש קרוב לשחקן, לפעמים המכה עברה “מעליו” או פספסה.

ב־V92 המכה משתמשת ב־capsule:

```text
מהשחקן
קדימה לכיוון המכה
עם radius של המכה
```

זה תופס גם אויבים שקרובים מאוד לשחקן.

### 3. התחמקות לפי הכפתור שנלחץ, לא לפי העכבר

עכשיו:

```text
Double W / Up    → dodge קדימה במסך
Double S / Down  → dodge אחורה
Double A / Left  → dodge שמאלה
Double D / Right → dodge ימינה
```

העכבר לא קובע את כיוון ההתחמקות.

## נשמר מ־V91

- Editor scene builder לא רץ בזמן Play Mode.
- Dodge effect הוא 2D ground trail.
- חיווי שאפשר לרפא את הסוס.
- WASD/חצים לא תלויים בעכבר.
- אין חרב על הסוס, רק ירי.
- קליעים פוגעים בקירות.
- Performance Guard.
- Enemy attack telegraphs.
- Ranged muzzle flash/trail/impact.
- Melee slash arcs.
- Enemy HP bars.
- Death reset.

## התקנה נקייה

מחק לגמרי:

```text
Assets/_Project
```

ואז העתק מה־ZIP את:

```text
Assets/_Project
```

חכה ל־Compile מלא ואז בדוק:
- סוס רחוק שחוזר אליך.
- שחקן שמתקרב לסוס עומד.
- מכה באויבים שנמצאים ממש צמוד לשחקן.
- דאבל־טאפ התחמקות לכל כיוון.
