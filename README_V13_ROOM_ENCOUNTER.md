# Boredom & Dungeons — V13 Room Encounter

V13 ממשיך את V12 ומוסיף את מערכת החדר הראשונה:

- חדר קרב עם קירות ופתחים
- אזורי יציאה
- Room Encounter Manager
- מעקב אחרי אויבים חיים
- זיהוי ניסיון בריחה דרך פתח
- Enemy Exit Blocker: אויב שרץ לפתח כדי לחסום בריחה
- הודעת Debug של מצב החדר
- סצנת בדיקה חדשה

## חשוב

V13 בנוי להמשיך אחרי V12.  
אם עדיין לא ייבאת את V12, תייבא קודם את V12 ואז את V13.

## התקנה

1. העתק את `Assets/_Project` מה־ZIP לתוך Unity.
2. חכה ל־Compile.
3. בתפריט לחץ:

```text
Boredom And Dungeons V13/Create Room Encounter Test Scene
```

4. לחץ Play.
5. לחץ בתוך Game/Simulator.

## שליטה

- WASD / Arrows — תנועה
- Space — קפיצה
- Left Shift — Dash
- C — Toggle Camera Follow
- Left Mouse / J — התקפה מהירה
- Right Mouse / K — התקפה חזקה

## מה לבדוק

1. יש חדר עם קירות ופתחים.
2. יש אויבים בתוך החדר.
3. כשאתה מתקרב ליציאה בזמן שיש אויבים חיים, מופיע ניסיון בריחה.
4. אויב ה־Exit Blocker רץ אל הפתח ומנסה לחסום אותך.
5. אם אתה הורג את כל האויבים, ה־Encounter מסתיים.
6. אחרי סיום הקרב, ניסיון יציאה כבר לא מפעיל חסימה.

## למה זה השלב הנכון

עד עכשיו בנינו:
- תנועה
- קפיצה
- Dash
- קרב
- 5 סוגי אויבים

עכשיו מתחילים להפוך את זה ל־gameplay room אמיתי, שזה הבסיס של המבוך.
