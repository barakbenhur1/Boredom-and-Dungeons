# Boredom & Dungeons — Clean Core V49

זו גרסה נקייה מלאה להחלפה, לא פאטץ׳.

## מה תוקן ב־V49

### 1. כפתורי הכיוון עובדים כמו שצריך

ב־V48 כל כפתור תנועה גרם לשחקן ללכת לאותו כיוון — לכיוון העכבר.  
זה לא נכון.

ב־V49 התנועה היא יחסית לכיוון העכבר:

```text
W / Up       = קדימה לכיוון העכבר
S / Down     = אחורה מהעכבר
A / Left     = סטרייפ שמאלה ביחס לעכבר
D / Right    = סטרייפ ימינה ביחס לעכבר
```

כלומר העכבר קובע את ה־forward, והמקשים קובעים את סוג התנועה ביחס אליו.

### 2. השחקן עדיין מסתובב במקום לפי העכבר

כאשר לא לוחצים תנועה:

```text
המודל מסתובב במקום לפי העכבר
השחקן לא זז
חרב/ירי יוצאים לכיוון העכבר
```

כאשר לוחצים תנועה:

```text
WASD / Arrows עובדים יחסית לכיוון העכבר
```

### 3. רגישות העכבר ירדה

הסיבוב כבר לא קופצני כל כך:

```text
Player rotation smoothing: 22 → 12
Player aim min distance: 0.45 → 1.25
Player aim smoothing: 10
Horse rotation speed: 32 → 12
Horse aim min distance: 0.75 → 1.45
Horse aim smoothing: 9
Camera follow/rotation smoothing: 24/32 → 16/18
```
